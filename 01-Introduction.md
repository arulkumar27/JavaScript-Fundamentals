# JavaScript Introduction

## What is JavaScript?

JavaScript is a high-level, lightweight, interpreted programming language used to create interactive and dynamic web applications. It runs directly inside web browsers and is also widely used on servers through Node.js.

JavaScript is one of the three core technologies of web development:

- HTML → Structure
- CSS → Styling
- JavaScript → Functionality

---

## Why Learn JavaScript?

JavaScript is essential for modern web development because it enables developers to:

- Build interactive websites
- Validate user input
- Fetch data from APIs
- Create web applications
- Develop backend services with Node.js
- Build mobile apps using React Native
- Automate repetitive tasks

---

## Real-world Example

When you click a "Login" button on a website and receive an error message without the page reloading, JavaScript is handling that interaction.

Examples include:

- Amazon shopping cart
- Netflix recommendations
- YouTube video controls
- Facebook notifications
- Gmail inbox updates
