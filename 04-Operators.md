# JavaScript Operators

## What are Operators?

Operators perform operations on values and variables.

---

## Arithmetic Operators

```javascript
let a = 10;
let b = 5;

console.log(a + b);
console.log(a - b);
console.log(a * b);
console.log(a / b);
console.log(a % b);
```

---

## Comparison Operators

```javascript
10 == "10"
10 === "10"
10 != 20
10 > 5
10 < 20
```

---

## Logical Operators

```javascript
&&
||
!
```

Example:

```javascript
let age = 20;

console.log(age > 18 && age < 60);
```

---

## Assignment Operators

```javascript
+=
-=
*=
/=
%=
```

---

## Increment

```javascript
count++;
```

---

## Decrement

```javascript
count--;
```

---

## Real-world Example

```javascript
let balance = 5000;

balance -= 1000;

console.log(balance);
```

---

## Best Practices

Use `===` instead of `==` to avoid unexpected type coercion.

---

## Summary

Operators are fundamental building blocks for calculations, comparisons, and decision-making.
